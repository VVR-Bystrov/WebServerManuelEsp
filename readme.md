# Простой HTTP-сервер.  
Сервер устанавливает состояние вывода GPIO по запросу  браузера.  
Формат   строки браузера:  
http://server_ip/gpio/0   
или  
http://server_ip/gpio/1  
Например , если в строке браузера  указать этот URL  
http://192,168,0,53/gpio/0   
то установит низкий уровень на выводе GPIO2.  
Если указать URL  
http:// 192,168,0,53/gpio/1  
то установит высокий уровень на выводе GPIO2  
Вэбсервер испытывался на платформе Wemos D1 mini  
Загрузка скетча производилась с применением Ардуино IDE  Версии 2.3.7  
Плата -Lolin(Wemos) D1 mini Pro  
Led pin -2;  
Для запуска WiFi сети необходимо в программе   
указать свои параметры  WiFi  сети:   ssid , password.  
Адрес IP Webservera нам сообщает скетч в мониторе порта.
В моем случае это был адрес 192.168.0.53 (приведен выше).  
Подключившись к сети WiFi, мы в строке браузера указываем 
его по формату, приведенному выше
